## Usage

### pubspec.yaml
```yaml
dependencies:
  webcrypto: ^0.5.5
  webcrypto_aurora:
    git:
      url: https://gitflic.ru/project/yacloud-ing/webcrypto_aurora.git
      ref: master
      path: ./
```

### Source code

Import `webcrypto.dart` in your Dart code:
```dart
import 'package:webcrypto/webcrypto.dart';
```
