## Usage

### pubspec.yaml
```yaml
dependencies:
  webcrypto: ^0.5.5
  webcrypto_aurora:
    git:
      url: https://github.com/TimoshN/webcrypto_aurora.git
      ref: main
      path: ./
```

### Source code

Import `webcrypto.dart` in your Dart code:
```dart
import 'package:webcrypto/webcrypto.dart';
```
